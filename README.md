# panda_robot_sim

Panda robot under NVIDIA isaac sim and ROS2.

------

## System Requirements

- OS : Ubuntu 24.04
- CPU : Intel Core i7 (7th Generation) / AMD Ryzen 5
- Cores : 4
- RAM : 32 GB
- Storage : 50GB SSD
- GPU : GeForce RTX 4080
- VRAM : 16GB
- Driver : Linux: 580.65.06

------

## Built with

- ROS Jazzy under Ubuntu 24.04 LTS
